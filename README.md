A beginner-friendly Python banking system that demonstrates core programming fundamentals such as functions, control flow, account operations, validation, and simple transaction logic.

## Features

- Create and manage customer account details
- Deposit and withdraw money
- Check account balance
- Practice menu-driven program design
- Demonstrates basic Python control flow and input validation

## Skills Demonstrated

- Python fundamentals
- Functions and modular code
- Conditional logic
- Loops
- Basic transaction handling

This is an early-stage programming project. I keep it public to show my learning progression from Python fundamentals toward full-stack development, data analytics, and AI/ML projects.
